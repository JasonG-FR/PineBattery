# PineBattery

## GTK app for monitoring the PinePhone, PinePhonePro and PineTab battery

### Dependencies

* python-gobject

### Install

Run the `install.sh` script to add the app to your launcher.

### Uninstall

Run the `uninstall.sh` script to remove the app from your launcher.
